# 无人机入侵植物三维重建与三维分割实验进度表

## 0. 课题总目标

**研究对象**：加拿大一枝黄花  
**任务主线**：
1. 2D 语义分割
2. 3D 重建 + 3D 语义渲染
3. 后续尺度恢复与体积估计
4. 服务于定量治理 / 精准喷洒

**当前方法主线**：
- 3D backbone：Scaffold-GS
- 3D 语义：在原有 MLP 基础上增加语义分支
- 2D 先验：Mask2Former + SAM2
  - Mask2Former：类别语义
  - SAM2：边界细化 + 连续帧一致性
- 后续目标：将 refined 2D pseudo-label 蒸馏到 3D 表达中

---

## 1. 当前核心结论总览

### 1.1 已有明确结论
- `sem_ramp` 是有效的
- 单独把 `start_semantic_iter` 延后到 5000，收益很小
- 当前 3D 主指标应以 **mIoU** 为主，PSNR 为保底
- 当前最值得验证的是：**densification 是否扰动几何并拖住 PSNR**
- 当前二维最值得推进的是：**Mask2Former + SAM2 两阶段协同**

### 1.2 当前最好结果
- 当前 3D 最优主结果：`exp01_sem_ramp`（test mIoU 0.7880，PSNR 24.71，SSIM 0.6902，LPIPS 0.2969）
- 当前最关键待验证实验：`exp03_late_sem_nodensify`

## B2. exp01_sem_ramp
- **实验名**：B2_exp01_sem_ramp
- **日期**：2026-04-18 ~ 2026-04-21
- **阶段**：3D
- **目的**：验证语义渐进加权是否有效
- **模型路径**：`output/dronev4_2_exp01_sem_ramp`
- **数据路径**：`data/dronev4_2`
- **输出内容**：3D 语义重建（语义渐进加权）
- **关键参数**：
  - start_semantic_iter: 0
  - mask_weight: 0.2
  - mask_warmup: 1000
  - mask_ramp: 3000
  - knn_weight: 0.05
  - knn_warmup: 2000
  - knn_ramp: 3000
  - update_from: 1500
  - update_interval: 100
  - update_until: 15000
  - load_iteration: 30000
- **中间测试结果**：
  - 7000 iter:
    - PSNR: 24.88
    - mIoU: 0.7809
  - 30000 iter:
    - PSNR: 24.71
    - mIoU: 0.7880
- **final evaluate**：
  - SSIM: 0.6902353
  - PSNR: 24.7041359
  - LPIPS: 0.2968660
- **现象观察**：相较 B1 baseline，30000 iter 的 test mIoU 从 0.7625 提升到 0.7880，说明语义渐进加权确实带来稳定收益。
- **初步结论**：当前 3D 最值得主推的一版。
- **下一步动作**：把这版作为主对照，继续与 B3/B4/B5 做消融对比，重点判断 late semantic 和 no_densify 的边际收益。
